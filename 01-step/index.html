Step 1

Add a app icon which will be shown on homesreen

First of all we need to create an icon. I used Medialoot's iOS 7 App Icon Kit to design one...

We need one icon for every possible resolution:

  • iPhone 60 x 60 px
  • iPhone Retina 120 x 120 px
  • iPad 72 x 72 px
  • iPad Retina 144 x 144 px

To let iOS know where your different icons are stored, simply add for each icon the following line to the <head>-section in your app. For a complete list check out the source of this page. 

<link rel="apple-touch-icon-precomposed" href="/path/to/your/icon" />

When you add this page to your homescreen, yo will see the icon.
