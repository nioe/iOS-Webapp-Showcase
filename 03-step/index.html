Step 3

Add a splash screen to your app which will be shown while the app is starting respectively your webpage is loading.

We need to create several images:

  • iPhone Portrait: 320 x 460 px
  • iPhone Retina Portrait: 640 x 920 px
  • iPhone 5 Portrait: 640 x 1096 px
  • iPad Portrait: 768 x 1004 px
  • iPad Landscape: 748 x 1024 px *
  • iPad Retina Portrait: 1536 x 2008 px
  • iPad Retina Landscape: 1496 x 2048 px *
* Important: Please note that the landscape images have to be rotated one time clockwise. Because of the information bar in iOS the resolution is different from the portrait images!
